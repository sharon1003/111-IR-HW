do i have windows xp service pack 2 installed  you can determine your service pack level using different methods click start run and type winver and press enter the following screen appears which shows the service pack level of your system if the words service pack 2 is present at the end of the line then you have windows xp sp2 installed alternately you can check the version information in the system properties sheet rightclick my computer icon in the desktop and choose properties the system tab should display the operating system and the service pack level of your installation or type msinfo32 in start run dialog the system summary sheet should display the information that you need
